## 🎯 Facial Attendance System

A real-time facial recognition attendance system built with Python. 
The application detects and identifies registered faces via webcam, 
then automatically logs each person's name along with a timestamp 
into an Excel sheet — no manual input required.

## ✨ Features
- Real-time face detection & recognition
- Auto-logs name + timestamp to Excel (.xlsx)
- Register new users with ease
- Duplicate entry prevention per session
- Lightweight and runs on standard hardware

## 🛠️ Tech Stack
- Python
- OpenCV — camera feed & face detection
- face_recognition — facial encoding & matching
- openpyxl / pandas — Excel logging

## 📦 Installation
pip install -r requirements.txt

## 🚀 Usage
python main.py
